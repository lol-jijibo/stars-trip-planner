# stars-trip-planner
**智能旅行助手项目简介**  **项目背景**：传统旅行规划面临**信息分散**（需频繁切换网站）、**缺乏个性化**（攻略千篇一律）以及**行程难调整**（牵一发而动全身）等三大痛点，导致行程规划耗时费力且准确度低。  **产品核心**：本项目打造的智能旅行助手，让用户只需输入简单诉求（如“北京3日游，喜好历史，预算中等”），即可**一站式自动生成个性化行程计划**（含景点、餐饮、住宿及预算）。同时系统支持**动态智能调整**，用户修改部分行程时，地图与预算会自动同步更新。  **技术价值**：这是一个真正落地的 Web 实用应用。通过该项目，开发者不仅能掌握将 AI 技术应用于实际业务的技巧，还能深度实践**多智能体（Multi-Agent）系统的设计与全栈 Web 应用的完整构建流程
